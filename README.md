In this project the LLM response is simply which tool to call and with which arguments here
Here the LLM does not start an external tool it simply decides the tool to use and arguments

At first the LLM goes and decides what tool to use with what argument
Then the tool searches and gives a output

At last the human query, tool used and the output from the tool are all fed into the LLM
This finally provides the desired output from all these information
